# Karma Build Script

Standardized build script for ktools projects.

## Usage
- Grab the latest git release: https://github.com/QM-Code/kbuild
- Copy [directory]/kbuild.py to an empty directory.
- Run ./kbuild.py --create-config --kbuild-root [directory]

Note: [directory] should be the relative or absolute path pointing to the pull directory.

See docs/kbuild.md for full documentation.
